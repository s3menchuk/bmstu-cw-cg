\chapter{Технологический раздел}

\section{Требования к ПО}

Разрабатываемое ПО должно обладать следующим функционалом:
\begin{itemize}
    \item добавление и удаление объектов;
    \item изменение положения, ориентации, масштаба и цвета объектов;
    \item изменение положения и ориентации камеры;
    \item добавление и удаление источников освещения;
    \item изменение положения, ориентации и цвета источников освещения;
\end{itemize}

Разрабатываемое ПО должно соответствовать следующим требованиями:
\begin{itemize}
    \item при запуске программы должна создаваться камера;
\end{itemize}

\section{Средства реализации}
Для написания данной курсовой работы был выбрал C++. Выбор языка программирования обусловлен следующими факторами:
\begin{itemize}
    \item быстродействие;
    \item поддержка ООП;
    \item богатый выбор графический библиотек.
\end{itemize}

В качестве графической библиотеке была выбрана SFML. Выбор графический библиотеке был обусловлен следующими факторами:
\begin{itemize}
    \item объектная модель (в отличие от glfw, sdl).
\end{itemize}

Для реализация графического интефейса пользователя был использован ImGui по причине легковесности, быстродействия и интеграции со всеми известными библиотекам.
Для генерации файлов сборки проекта был использован CMake.
