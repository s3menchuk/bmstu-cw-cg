\section{Общий алгоритм построения изображений}
В качестве средства описания общего алгоритма построения изображений будет использоваться диаграмма IDEF0.

Построение изображений в компьютерной графике в контексте диаграммы IDEF0 определяется следующими параметрами:

Описание элементов входных данные:
\begin{itemize}
    \item Объекты определяются формой, цветом, материалом, отражающими характеристики;
    \item Камера определяется расположением и ориентацией;
    \item Источники освещения определяются типом, расположением и направлением (при наличии), цветом и интенсивностью;
\end{itemize}

Описание элементов управление:
\begin{itemize}
    \item Алгоритм рендеринга: растеризация, трассировка лучей, path tracing, ray marching;
    \item Модель освещения: локальная или глобальная;
    \item Способ закраски: простая, по Гуро, по Фонгу.
\end{itemize}

Описание механизмов:
\begin{itemize}
    \item Аппаратное обеспечение (ЦП, ОЗУ, ГП);
    \item Программное обеспечение;
    \item Язык программирования (C++, Python, C\#, Java и др.);
    \item Графический API (OpenGL, Vulkan, DirectX, Metal и др.);
    \item Графическая библиотека/фреймворк (GLFW, SDL, SFML и др.)
\end{itemize}

Выходными данными является сгенерированный кадр.
