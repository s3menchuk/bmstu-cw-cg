	\chapter{Флеш-накопитель с разработранным ПО}
	Файлы на флеш-накопители имеют следующую структуру:
	\begin{itemize}
		\item src -- файлы исходным кодом;
		\item include -- заголовочные файлы;
		\item CMakelists.txt -- файл сборки проекта;
		\item docs -- документация (ТЗ, РПЗ, презентация, календарный план);
		\item assets -- модели (*.obj);
		\item images -- сгенрированные изображения;
		\item .clang-format -- файл с настроками форматирования кода;
		\item README.md -- файл с общим описанием проекта;
		\item build -- каталог для сборки проекта по умолчанию;
		\item .gitignore -- файлы, исключенные из системы контроля версий.
	\end{itemize}
\end{appendices}
