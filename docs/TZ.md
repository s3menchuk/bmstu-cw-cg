# Техническое задание (ТЗ)

Разработать программное обеспечение с графическим интерфейсом, позволяющее отображать (визуализировать) сцену, состоящую из геометрических примитивов (шар, параллелепипед {куб}, призма, пирамида, цилиндр, конус ...). Программа должна позволять добавлять новые примитивы, камеры, источники освещения. А также изменять их положение в пространстве и вращать по всем осям.
